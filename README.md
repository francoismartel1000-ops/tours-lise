# Tours Lise B. v2 — Instructions

## Fichiers
- `index.html`, `style.css`, `app.js`, `data.js` — Application
- `manifest.json`, `sw.js` — PWA
- `icon-192.png`, `icon-512.png` — Icônes

## Déploiement GitHub Pages

1. Créer compte sur **github.com**
2. Nouveau dépôt : `tours-lise` (Public)
3. Uploader TOUS les fichiers
4. Settings → Pages → Branch: main → Save
5. Lien : `https://VOTRE-NOM.github.io/tours-lise/`

## Installation iPhone (Safari uniquement)

1. Ouvrir le lien dans **Safari**
2. Icône ⬜↑ Partager → **"Sur l'écran d'accueil"**
3. **Ajouter** ✅

## Utilisation

### Saisons
- Bouton **Saison 2025** en haut → changer ou créer une saison
- Chaque saison a ses propres tours

### Tours (saisie)
- **＋ Nouveau tour** pour ajouter
- Chaque tour contient : infos + pourboires + lignes de travail
- **📄** = ajouter une photo de feuille de travail
- **›** = modifier le tour

### Pourboires & Travail
- Vues lecture seule, générées automatiquement depuis les tours
- Pour modifier : aller dans l'onglet Tours

### Lignes de temps
- Chaque tour peut avoir plusieurs lignes
- Le temps réel (fin - début) s'affiche automatiquement
- Le montant suggéré est calculé depuis le dernier taux de la saison
